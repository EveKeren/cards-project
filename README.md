# BCard (React + Vite)

Small app to browse, create, edit, delete, and like business cards.

## Stack

React 19 • React Router 7 • Vite 6 • MUI 7 • Axios • Joi • jwt-decode • ESLint

## Setup

```bash
npm install
npm run dev
# build: npm run build | preview: npm run preview | lint: npm run lint
```
